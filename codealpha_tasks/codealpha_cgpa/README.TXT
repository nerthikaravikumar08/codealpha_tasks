Projectname: CGPA Calculator 

Internship: CodeAlpha C++ Programming Internship

Description:
This project is a simple CGPA Calculator developed using C++.
The program calculates the CGPA of a student by taking the grade
points and credit hours for each subject as input.

Features:

Takes number of subjects from the user

Accepts grade points for each subject

Accepts credit hours for each subject

Calculates and displays the final CGPA


How the Program Works:

1. The user enters the total number of subjects.


2. The program asks for grade points and credit hours for each subject.


3. It multiplies grade points with credit hours to calculate total points.


4. Finally, CGPA is calculated using the formula:



CGPA = Total Grade Points / Total Credit Hours

Programming Language Used:
C++
