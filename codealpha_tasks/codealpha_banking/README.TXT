

Project Name: Banking System

Internship: CodeAlpha C++ Programming Internship

Description:
This project is a simple Banking System developed using C++.
The program simulates basic banking operations such as creating
an account, depositing money, withdrawing money, and displaying
account details.

Features:

Create a bank account

Deposit money into the account

Withdraw money from the account

Display account details including name, account number, and balance


How the Program Works:

1. The user creates a bank account by entering name, account number, and initial balance.


2. The program shows a menu with options like deposit, withdraw, display details, and exit.


3. The user selects an option to perform the desired banking operation.


4. The program updates and displays the account information accordingly.



Programming Language Used:
C++
